# Sheets Dashboard AI

Дашборд для анализа Google Sheets через Claude AI.

## Деплой на Vercel (5 минут)

### Шаг 1 — Загрузи код на GitHub
1. Зайди на github.com → кнопка «New repository»
2. Назови репозиторий `sheets-dashboard`
3. Загрузи все файлы из этой папки

### Шаг 2 — Задеплой на Vercel
1. Зайди на vercel.com → «Add New Project»
2. Выбери свой репозиторий `sheets-dashboard`
3. Нажми «Deploy» (настройки менять не нужно)

### Шаг 3 — Добавь API ключ
1. В Vercel открой Settings → Environment Variables
2. Добавь переменную:
   - Name: `ANTHROPIC_API_KEY`
   - Value: твой ключ с console.anthropic.com
3. Нажми Save → зайди в Deployments → Redeploy

### Готово!
Твоё приложение доступно по адресу `https://sheets-dashboard-xxx.vercel.app`

## Как получить API ключ
1. Зайди на console.anthropic.com
2. Зарегистрируйся
3. API Keys → Create Key
4. Скопируй ключ (он начинается с `sk-ant-...`)

## Как подготовить Google Sheets
1. Открой таблицу
2. Файл → Поделиться → Настройки доступа
3. Измени на «Все у кого есть ссылка могут просматривать»
4. Скопируй ссылку и вставь в приложение

## Структура таблицы
Первая строка — заголовки столбцов.
Первый столбец — метки (месяцы, категории и т.д.).
Остальные столбцы — числовые данные.

Пример:
| Месяц | Продажи | Расходы | Прибыль |
|-------|---------|---------|---------|
| Янв   | 820000  | 540000  | 280000  |
| Фев   | 740000  | 510000  | 230000  |
